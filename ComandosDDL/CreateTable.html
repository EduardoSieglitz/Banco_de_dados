Aprenda a Usar CREATE TABLE no SQL

O que é o comando CREATE TABLE?

O comando CREATE TABLE no SQL é usado para criar uma nova tabela em um banco de dados. É necessário definir o nome da tabela e as colunas com seus respectivos tipos de dados.

Exemplo Básico

O código abaixo cria uma tabela chamada clientes com três colunas:

CREATE TABLE clientes (
    id INT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100)
);
